Gritter notification support for Nette Framework
================================================
Tomas Kraina (twitter.com/krajaac)


For more information about Gritter check Jordan Boesch's blogpost - http://boedesign.com/blog/2009/07/11/growl-for-jquery-gritter/ or Gritter's github repo - http://github.com/jboesch/Gritter.


Installing
----------

This demo application is based on nette application skeleton.

Make sure that directories 'app/temp' and 'app/log' writable.


Requirements
------------

Requires PHP 5.2.0 or higher
